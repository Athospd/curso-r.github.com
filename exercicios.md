### Flow Controls

1.a) O que acontece se eu rodar


```r
x <- 4
if(x = 4) {
  'isso aqui apareceu'
}
x
```

1.b) Como você modificaria esse código para não dar erro e aparecer a frase 'isso aqui apareceu'?
